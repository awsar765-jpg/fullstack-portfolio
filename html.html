What is HTML?

HTML (HyperText Markup Language) is the foundation of every website. It is used to define the structure and semantic meaning of web content using various tags and elements.

In this demo, I have created a clean student profile page using proper semantic HTML5 elements.

What This Demo Demonstrates

  • Use of required attribute for basic validation
